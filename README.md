# UPI Transaction Data Analysis Dashboard
This repository contains a fully interactive Power BI dashboard for analyzing UPI transactions. It offers a powerful suite of filters and rich visualizations to explore transactional behavior, patterns, and key financial metrics.

# Features
Multifaceted Slicers: Filter data by Bank Sent, Bank Received, City, Device Type, Gender, Age Group, Merchant Name, Payment Method, Purpose, and Transaction Type for highly customizable analysis.

# Dynamic Visualizations:

Line Charts: Track monthly trends for both transaction amounts and balances.

Column Charts: Compare transaction volumes across different months at a glance.

Aggregated Tables: View granular monthly breakdowns alongside dynamic totals for deeper insight.

Bookmarks & Toggle Buttons: Easily switch between different views and charts to focus on amounts or balances as per business needs.

Intuitive Layout: Sleek and professional design, ensuring both accessibility for business users and depth for data analysts.

# Getting Started

Download or clone the repository.

Open the Power BI dashboard file in Microsoft Power BI Desktop.

Replace the sample data with your dataset or connect to your data source.

Use the slicers and buttons to start exploring your UPI transaction data.
